h1. Tpett's Janus Configuration

h3. Description

These are my local "Janus":https://github.com/carlhuda/janus configuration files.  These are very specific to my personal preferences.  I recommend you fork this repository so you can customize to taste.

h3. Installation

bc. $ git clone git://github.com/tpett/janus.local.git ~/.janus.local
$ cd ~/.janus.local
$ rake

The rake command will add the appropriate symlinks to these files so Janus will find them.
